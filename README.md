# ⚖️ AI Legal Notice Generator

An AI-powered web application that generates professional legal notices using Streamlit and LangChain.

---

## 🚀 Features

- Generate legal notices for:
  - Rental Disputes
  - Cheque Bounce
  - Property Issues
  - Payment Recovery
- Structured legal format
- English and Tamil language support
- Download generated notice

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- OpenAI API
